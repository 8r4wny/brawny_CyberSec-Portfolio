# Home Network Asset Classification

This project classifies assets connected to a typical home network using an Excel spreadsheet.
It focuses on basic asset identification and cybersecurity awareness.

## What This Project Shows
- Understanding of what an asset is
- Basic asset classification
- Awareness of security context

## Tools Used
- Microsoft Excel

## Files
- Home_Network_Assets.xlsx

## Notes
This project uses hypothetical data and contains no sensitive information.
